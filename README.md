## Setup Instructions

To get started with this repository, follow these steps:

1. **Use Visual Studio Code**: Clone the repository and open it in VS Code. Make sure to open the workspace file at the root of the repo (`spotmicroai.code-workspace`).

2. **Accept Recommended Plugins**: When prompted, accept the recommended extensions, which are primarily related to Python development.

3. **Create Environment**: Open the Command Palette (Ctrl+Shift+P) and select "Create Environment" to set up your Python environment.

4. **Install Dependencies**: Run the following command to install the development requirements:

   ```
   pip install -r requirements-dev.txt
   ```
